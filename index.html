What This Is

This is an unofficial local dashboard made by affiliates who were tired of the Klook Kreator dashboard experience and wanted better visibility into commission, refunds, destinations, and audience signals.

What This Is Not

This is not a Klook product. It is not endorsed, supported, or maintained by Klook. It is not official reporting software, tax software, accounting software, or financial advice.

Privacy And Data Handling

Your CSV imports, backup files, and derived dashboard data stay on your computer. The app stores imported data locally in your browser using IndexedDB.

This dashboard does not upload your booking data, customer data, or reports to any server controlled by this project. There is no backend for this app.

Internet Requests

The dashboard currently loads a few JavaScript libraries from CDNs so the app can run in the browser:

  • cdn.jsdelivr.net for Papa Parse, used to read CSV files in the browser.
  • cdn.jsdelivr.net for Chart.js, used for charts.
  • cdn.plot.ly for Plotly, used for the country map.

Those requests fetch library files only. The dashboard code does not send your imported CSV contents or saved dashboard data to those CDNs.

If you want zero third-party network requests at runtime, these libraries can be vendored locally instead of loaded from CDNs.
